# DBR進捗管理 v1.1 マニュアル

## 概要
`DBR進捗管理` は、DBR譜面のクリア状況、BP、スコア、メモをローカルで管理するためのWebアプリです。  
データはブラウザの `localStorage` に保存されます。

## 画面構成
- `ヘッダー部`
  データ入出力ボタンがあります
- `全体サマリー`
  難易度帯ごとの横棒グラフと、クリアランプ別の件数を表示します
- `曲一覧`
  現在の絞り込み条件に一致する曲カードを表示します
- `プレー記録`
  選択曲に対してクリア、BP、スコア、メモを入力します
- `推移グラフ`
  BP推移とスコア推移を表示します
- `選択曲の履歴`
  保存済み履歴を日付順で表示します
- `絞り込み`
  フローティングの簡易絞り込みです

## 初期準備
### 1. 難易度表を読み込む
- `難易度表読み込み` を押します
- 曲一覧は難易度表データを基準に構築されます

### 2. 既存データを取り込む
- `CSV読み込み`
  縦データの履歴CSVを読み込みます
- `JSON読み込み`
  DBR難易度表対応形式のJSONを読み込みます

## 曲一覧の見方
- 曲カードには以下が表示されます
  - 難易度
  - SPLv.
  - ベストクリアランプ
  - Best BP
  - Latest BP
  - 最終プレー日
- カードを押すと、その曲が `プレー記録` に反映されます

## プレー記録の付け方
### 1. 曲を選ぶ
- 曲一覧から曲カードを押します

### 2. 入力する
- `クリア`
- `BP`
- `スコア`
- `メモ`

### 3. 保存する
- `保存する` を押します
- 同一曲・同一日の記録がある場合は、その日の記録を更新します

### 4. メモ
- メモは曲ごとに保存されます
- メモ欄からフォーカスが外れた時に自動保存されます
- メモだけを保持することもできます

### 5. 当日の記録削除
- `当日の記録を削除` を押すと、選択曲の今日の記録だけ削除します

### 6. 戻る
- `戻る` を押すと、選択中の曲カード位置へスクロールします

## 全体サマリー
### 横棒グラフ
- 現在の絞り込み軸に応じて、難易度帯ごとの内訳を表示します
- `Lv.`
- `SPLv.`
- `片手Lv.`
- 棒の長さは一定で、内部をクリアランプ比率で積み上げています

### クリアランプ枠
- `NO PLAY` などの枠を押すと、曲一覧側のクリアランプ絞り込みを切り替えます
- 最後の1個を無効化しようとした場合は全有効に戻ります
- PCでは、同じ枠を短時間で2回押すと、そのランプだけ有効な状態にできます
- スマホでは、枠を左にスワイプすると、そのランプだけ有効な状態にできます
- 件数表示は、現在の絞り込み条件に追従します
- `曲名` または `メモ` で絞り込んでいる間は、クリアランプ枠の操作は無効になります

### フィルタ表示
- `フィルタを表示` から固定フィルタを開けます
- 対応項目:
  - `INFINITAS`
  - `AC収録`
  - `未査定曲`
  - `おすすめ`
  - `リセット`
- `リセット` は固定フィルタだけを初期化します
  - `INFINITAS`
  - `AC収録`
  - `未査定曲`
  - `おすすめ`
- `絞り込み軸`、その値、クリアランプ状態は維持されます

## フローティング絞り込み
### 絞り込み軸
- `Lv.`
- `SPLv.`
- `片手Lv.`
- `曲名`
- `メモ`

### 数値軸
- スライダーで単一値を選択します
- スライダーを離した時に確定します

### 曲名
- 曲名の部分一致で検索します
- フォーカスアウト時に検索が確定します
- `解除` ボタンで直前の絞り込み状態へ戻せます

### メモ
- メモの部分一致で検索します
- フォーカスアウト時に検索が確定します
- `解除` ボタンで直前の絞り込み状態へ戻せます
- 曲名検索欄とメモ検索欄の入力内容は、それぞれ別に復元されます

## 並び順
- `曲名`
- `Lv.`
- `SPLv.`
- `片手Lv.`
- `最終プレー`
  - 最終プレー日の昇順です
  - 履歴なしの曲は先頭に並びます
- `クリアランプ`

## データ入出力
### CSV読み込み
- 縦データCSVを読み込みます
- 同一 `title + date` の行は上書きされます
- `memo` 列があれば曲メモも復元します

### CSVを書き出す
- 現在の履歴を縦データCSVで書き出します
- 出力列:
  - `date`
  - `title`
  - `level`
  - `lamp`
  - `bp`
  - `score`
  - `memo`
- 記録のない曲でも、メモだけある場合は `memo` 専用行として出力されます

### JSON読み込み
- DBR難易度表対応形式のJSONを読み込みます
- 読み込み時に記録日を指定できます
- 同一 `title + date` の既存記録は置き換えます

### JSONを書き出す
- DBR難易度表対応形式のJSONを書き出します
- 主に外部連携用です

### データ全削除
- 保存済みのプレー記録を全削除します
- 難易度表データは残ります

## 注意点
- 曲一覧は難易度表データを基準にしています
- `片手Lv.` を絞り込み軸にした場合、片手難易度未設定曲は一覧と集計から除外されます
- `曲名` または `メモ` を絞り込み軸にした場合、クリアランプ枠の操作は無効になります
- データ保存先はブラウザごとの `localStorage` です
- ブラウザや端末を変えると、保存データは自動では共有されません

## 推奨運用
- 履歴管理の主データは `CSV` を使う
- 外部連携や難易度表対応データ交換には `JSON` を使う
- 作業前後で `CSV` または `JSON` を書き出してバックアップを取る
