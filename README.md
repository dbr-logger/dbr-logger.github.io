# DBR進捗管理 v1.2 マニュアル

## このアプリについて
`DBR進捗管理` は、DBR難易度表との正式連携・共同開発により制作している、DBR譜面の記録管理アプリです。

DBR難易度表のデータをもとに曲一覧を作り、クリアランプ、BP、スコア、メモをまとめて管理できます。片手Lv.には、片手難易度表から提供いただいたデータを利用しています。

日々のプレー記録はCSVでバックアップでき、DBR難易度表対応形式のJSONとして書き出してDBR IR連携にも利用できます。

記録データはブラウザの `localStorage` に保存されます。端末やブラウザをまたいだ自動同期はありません。大切な記録は、定期的にCSVで書き出しておくと安心です。

## はじめかた
1. 難易度表を読み込む  
   上部の `難易度表読み込み` を押します。未読み込み、または前回の読み込みから12時間以上経っている場合は、ボタンが目立つ色で表示されます。
2. 曲一覧を確認する  
   読み込みが終わると、DBR難易度表のデータをもとに曲一覧が表示されます。片手Lv.がある曲では、その情報もあわせて確認できます。
3. 必要に応じて記録を取り込む  
   以前の記録がある場合は、CSVまたはJSONから取り込めます。

難易度表を読み込むと、保存済み記録の曲名表記や譜面識別情報が、難易度表の情報をもとに更新される場合があります。

## 記録する
1. 曲を選ぶ  
   曲一覧から曲カードを選びます。選んだ曲の入力欄、推移グラフ、履歴が表示されます。
2. プレー内容を入力する  
   クリアランプ、BP、スコア、メモを入力できます。BPはそのまま入力するほか、BAD/POORに分けて入力する表示にも切り替えられます。
3. 保存する  
   `保存する` を押すと、クリアランプ、BP、スコアのいずれかが入力されている場合にプレー履歴として保存されます。

### BP入力
- 通常はBPを直接入力します。
- `BAD/POOR入力` に切り替えると、BADとPOORを分けて入力できます。
- BAD/POOR入力では、BADとPOORの合計をBPとして保存します。
- BADとPOORのどちらか一方だけが入力されている場合は保存できません。

### メモ
- メモは曲ごとに保存されます。
- メモだけを入力して保存すると、プレー履歴は追加せず、メモだけを更新します。
- メモ欄を空にして保存すると、メモを削除できます。

## 曲一覧を見る
曲カードでは、譜面情報と自分の記録をまとめて確認できます。カード左側の色は、現在のクリアランプに対応しています。

| 表示 | 内容 |
| --- | --- |
| 難易度 / SPLv. / 片手Lv. | DBR難易度表の情報と片手Lv.です。 |
| クリアランプ | 保存済み記録から見た現在のクリア状況です。 |
| Best BP | これまでの記録の中で最も少ないBPです。 |
| Latest BP | 最新プレーのBPです。 |
| 最終プレー日 | 最後に記録した日付です。 |
| 履歴件数 | 保存されているプレー履歴の件数です。 |
| おすすめ度・メモ | 曲名の下に表示されます。 |

新規提案中の譜面は、曲カードが黄色背景になり、`新規提案中` バッジが表示されます。

## 絞り込む
画面上に表示されるQUICKFILTERから、曲一覧の絞り込み条件を選べます。絞り込み軸ごとに、前回選んだ条件が復元されます。

### 絞り込み軸
| 軸 | 使い方 |
| --- | --- |
| Lv. | DBR難易度で絞り込みます。単一選択と範囲選択を使えます。 |
| SPLv. | SPLv.で絞り込みます。単一選択と範囲選択を使えます。 |
| 片手Lv. | 片手難易度で絞り込みます。片手難易度がない曲は一覧と集計から外れます。 |
| プレー日 | 記録した日付で絞り込みます。単一の日付と範囲指定を使えます。 |
| 曲名 | 曲名で検索します。表記ゆれや一部の記号違いも吸収して検索します。 |
| メモ | 保存済みメモの内容で検索します。 |

### 数値軸の単一選択・範囲選択
- Lv.、SPLv.、片手Lv.では、スライダーで値を選びます。
- `範囲選択` を押すと、2点のスライダーで範囲を指定できます。
- `単一選択` を押すと、1つの値を選ぶ表示に戻ります。
- 単一選択と範囲選択の状態は、Lv.、SPLv.、片手Lv.ごとに保持されます。

### プレー日
- 単一選択では、1つの日付の記録を表示します。
- 範囲選択では、開始日と終了日を指定して表示します。
- 日付は履歴がある日付を基準に操作できます。
- 範囲選択で前回の範囲がない場合は、履歴がある日のうち直近14日分が選択されます。
- 履歴がない場合は、今日の日付が入ります。
- `戻す` を押すと、既定の日付または範囲に戻せます。

### 固定フィルタ
- 曲データ、未査定曲、おすすめ、譜面難易度の条件を設定できます。
- 曲データでは、すべて、AC、INFINITAS、ACのみ、INFINITASのみ、CS/削除曲を選べます。
- 譜面難易度では、B、N、H、A、Lを選べます。
- ACのみ、INFINITASのみは片方だけに収録されている曲、CS/削除曲はどちらにも収録されていない曲です。
- `リセット` は固定フィルタだけを初期化します。
- 曲名またはメモで絞り込んでいる間は、クリアランプフィルタの操作は無効になります。
- 曲名またはメモで絞り込んでいる間は、未査定曲、おすすめ、譜面難易度の固定フィルタは曲一覧に反映されません。
- プレー日で絞り込んでいる間は、曲データ、未査定曲、おすすめ、譜面難易度の固定フィルタは曲一覧に反映されません。

## 並び替える
曲一覧上部の並び順から、表示順を変更できます。`▲` / `▼` で昇順・降順を切り替えられます。

選べる並び順は、曲名、Lv.、SPLv.、片手Lv.、クリアランプ、最小BP、最新BP、最終プレー、プレー回数、おすすめ、メモです。

並び順は絞り込み軸ごとに保持されます。絞り込み軸を変えた時は、昇順で表示されます。

## 全体サマリーを見る
全体サマリーでは、現在の絞り込み条件に合わせて、難易度帯ごとの内訳とクリアランプ別の件数を確認できます。

### 帯グラフ
- Lv.、SPLv.、片手Lv.では、難易度帯ごとのクリアランプ内訳を表示します。
- プレー日では、1本の帯が1日の記録を表します。
- プレー日がALLの場合は、履歴がある日のうち直近14日分を表示します。
- プレー日の範囲を指定している場合は、その範囲内で履歴がある日を表示します。

### クリアランプフィルタ
- `NO PLAY` などの枠を押すと、曲一覧側のクリアランプ絞り込みを切り替えます。
- PCでは、ダブルクリックすると、そのランプだけ有効な状態にできます。
- スマホでは、枠を左にスワイプすると、そのランプだけ有効な状態にできます。

## 推移グラフと履歴を見る
曲カードを選ぶと、その曲のBP推移、スコア推移、保存済み履歴が表示されます。

- 履歴には、日付、クリアランプ、BP、スコアが表示されます。
- 履歴は個別に削除できます。
- プレー日で絞り込み、かつ最終プレーで並び替えている場合は、同じ曲の複数履歴が曲一覧に表示されることがあります。

## 提案する
選択中の曲カードには、譜面の状態に応じて提案ボタンが表示されます。

| ボタン | 送信できる内容 |
| --- | --- |
| 新規提案 | レベル、おすすめ度、コメント、備考を送信できます。 |
| 変更提案 | 変更後レベルと提案理由を送信できます。 |
| おすすめ提案 | 変更後おすすめ度と提案理由を送信できます。 |

新規提案のコメントは表に反映されます。備考は提案内容の補足として送信されますが、表には反映されません。

送信後は、送信先の外部スプレッドシートを開いて内容を確認できます。すでに提案中の譜面だった場合は、提案内容を確認できるリンクが表示されます。

## データを読み込む・書き出す
履歴全体の保管や移行にはCSVを使うのがおすすめです。DBR IRへ記録を反映したい場合は、JSONファイルを手元で扱うよりも `DBR IR送信` を使うとスムーズです。

JSON読み込み・JSON書き出しは、DBR難易度表対応形式のデータをファイルとして扱いたい場合に使用します。

### CSV読み込み
- DBR進捗管理が書き出した履歴CSVを読み込みます。
- 同一 `textageKey + timestamp`、または同一 `title + timestamp` の行は上書きされます。
- `lamp`、`bp`、`score` の一部が欠けていても取り込めます。
- `memo` 列があれば、曲メモも復元します。
- `timestamp` 列があれば、同じ日の履歴順も復元します。

### CSV書き出し
現在の履歴をCSVで書き出します。出力列は次の通りです。

`timestamp` / `date` / `textageKey` / `title` / `level` / `splv` / `lamp` / `bp` / `score` / `memo`

- `level` はDBR難易度表の難易度を出力します。
- `splv` はDBR難易度表のSPLv.を出力します。
- 記録のない曲でも、メモだけある場合は `memo` 専用行として出力されます。

### JSON読み込み
- DBR難易度表対応形式のJSONを読み込みます。
- 読み込み時に記録日を指定できます。空欄の場合は今日の日付として読み込みます。
- ランプ、BP、スコアのいずれかが記録日以前の最良記録を更新する場合のみ、履歴として取り込まれます。
- すでに反映済みの項目は取り込まず、未反映のランプ、BP、スコアだけを取り込みます。

### JSON書き出し
DBR難易度表対応形式のJSONをファイルとして書き出します。DBR IRへ反映したい場合は、次の `DBR IR送信` を使います。

- BPは最小値、スコアは最大値、クリアランプは最上位ランプを出力します。
- 書き出し時は、難易度表を再読み込みしてから書き出します。
- 難易度表の読み込みに失敗した場合は、現在保持している難易度表データで書き出します。

### DBR IR送信
DBR IRへ記録を反映したい場合に使用します。DBR難易度表対応形式のJSONデータをDBR IR受け取りページへ渡します。

送信時は、難易度表を再読み込みしてから実行します。

### データ全削除
保存済みのプレー記録とメモを全削除します。難易度表データは残ります。

この操作は取り消せません。削除前にCSVで書き出しておくことをおすすめします。

## キーボードで操作する
PCでは、キーボードからQUICKFILTERや曲カード選択を操作できます。

| キー | 操作 |
| --- | --- |
| `Q` | QUICKFILTERを開閉します。 |
| `F` / `W` / `E` | Lv.、SPLv.、片手Lv.へ移動します。 |
| `R` | プレー日へ移動します。 |
| `S` / `T` | 曲名検索、メモ検索へ移動します。 |
| `A` / `D` | スライダーや日付を操作します。 |
| `Shift + A` / `Shift + D` | 範囲選択の右側ポイントや終了日を操作します。 |
| 同じ軸のショートカット | Lv.、SPLv.、片手Lv.、プレー日では、単一選択と範囲選択を切り替えます。 |
| 数字キー / テンキー | 表示中の曲カードを選択します。 |
| `Delete` | 曲名またはメモ絞り込みを開いている時に、検索文字を消して検索欄をアクティブにします。 |
| `Esc` | 入力欄のフォーカスを外します。検索や日付絞り込みは確定しません。 |

入力欄を操作している間は、ショートカットは動作しません。

## 推奨運用
| 用途 | 内容 |
| --- | --- |
| バックアップ | 履歴管理の主データはCSVで保管すると扱いやすいです。 |
| 外部連携 | DBR IRへ反映したい場合は、DBR IR送信を使います。 |
| 難易度表の更新 | 定期的に `難易度表読み込み` を押して、曲情報を新しい状態に保ちます。 |
| 記録の保護 | ブラウザや端末を変える前に、CSVを書き出しておくと安心です。 |

## 注意事項
- 曲一覧はDBR難易度表データを基準にしています。
- 片手Lv.で絞り込んだ場合、片手難易度が未設定の曲は一覧と集計から除外されます。
- CSVやJSONの取り込み時に `textageKey` がない場合、曲名表記が一致していないと同じ譜面として扱えない場合があります。
- テーマ設定はブラウザの `localStorage` に保存されます。
- データ保存先はブラウザごとの `localStorage` です。ブラウザや端末を変えると、保存データは自動では共有されません。
- このアプリは、各難易度表を置き換えるものではなく、それらを参照しながら個人のプレー記録を管理しやすくするための補助ツールです。
